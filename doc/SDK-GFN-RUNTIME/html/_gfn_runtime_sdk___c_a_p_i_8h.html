NVGFNSDK_EXPORT GfnRuntimeError NVGFNSDKApi gfnRegisterSaveCallback ( SaveCallbackSig  saveCallback,
void *  pUserContext 
)
Description
Register an application callback with GFN to be called when GFN needs the application to save user progress. It is recommended that this be implemented as an autosave if such a feature is supported by your application.
Usage
Register an application function to call when GFN needs the application to save
Parameters
saveCallback- Function pointer to application code to call when GFN needs the application to save
pUserContext- Pointer to user context, which will be passed unmodified to the callback specified. Can be NULL.
Return values
gfnSuccess- On success when running in a GFN environment
gfnDLLNotPresent- If callback was not registered
