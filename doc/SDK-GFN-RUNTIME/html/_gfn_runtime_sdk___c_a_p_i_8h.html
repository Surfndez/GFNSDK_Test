Enumerator

Return to indicate that a callback has performed the requested operation.

Return to indicate that a callback did not perform the requested operation.

enum GfnIsRunningInCloudAssurance

Returned from gfnIsRunningInCloudSecure to denote security assurance that the calling application is running, untampered, in the GFN cloud environment.

Enumerator
gfnNotCloud 

Not considered to be running in GFN cloud, as it looks like a client/local system.

gfnIsCloudLowAssurance 

Considered to be running in GFN Cloud, using software heuristics that are not guaranteed against circumvention.

gfnIsCloudMidAssurance 

Considered to be running in GFN Cloud, using software and network heuristics that are difficult to circumvent.

gfnIsCloudHighAssurance 

Considered to be running in GFN Cloud, using software and hardware heuristics that are near impossible to circumvent.

enum GfnStreamStatus

Enumerator

Initial default state.

GfnStreamStatusNetworkTest 

Client is running a network test to determine the best zone.

GfnStreamStatusLoading 

Client is loading the game.

GfnStreamStatusStreaming 

Client is actively streaming.

GfnStreamStatusDone 

Client has successfully finished streaming.

GfnStreamStatusError 

Client has stopped streaming and has entered an error state.

GfnStreamStatusGotInputFocus 

Client has gained input focus to the streamer window.

GfnStreamStatusLostInputFocus 

Client has lost input focus to the streamer window.

NVGFNSDK_EXPORT GfnRuntimeError NVGFNSDKApi gfnRegisterClientInfoCallback ( ClientInfoCallbackSig  clientInfoCallback,
void *  pUserContext 
)
Description
Register an application callback with GFN to be called when certain client info that is part of GetClientInfo API changes
Usage
Register an application function to call when a the client information from the GFN user's client system has changed
Parameters
clientInfoCallback- Function pointer to application code to call when client information has changed
pUserContext- Pointer to user context, which will be passed unmodified to the callback specified. Can be NULL.
Return values
gfnSuccess- On success when running in a GFN environment
gfnInvalidParameter- If callback was NULL
gfnCloudLibraryNotFound- If the on-seat dll was not present (Usually due to not running on a seat)
gfnAPINotFound- If the API was not found in the GFN SDK Library
gfnCallWrongEnvironment- If the on-seat dll detected that it was not on a game seat
